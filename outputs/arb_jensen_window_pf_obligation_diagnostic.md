## What Was Checked

For each fixed `lambda`, shift `n`, and degree `d`, the probe forms the
banded Toeplitz matrix:

```text
T(r,c) = B^{d,n,lambda}_{c-r}
```

with entries outside `0..d` set to zero. It then checks the contiguous minors:

```text
det T[rows 0..m-1, cols 1..m]
```

for the selected sizes.
