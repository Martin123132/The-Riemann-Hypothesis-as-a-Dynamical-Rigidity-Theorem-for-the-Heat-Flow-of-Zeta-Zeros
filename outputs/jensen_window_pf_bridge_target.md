# Jensen-Window PF Bridge Target

Date: 2026-07-05

Status: theorem target artifact. This is not a proof of PF-infinity,
Laguerre-Polya membership, RH, or `Lambda <= 0`; it is a sharper formulation
of the signed-Hankel/Jensen bridge obligation.

## Purpose

The signed-Hankel route must ultimately prove hyperbolicity of every Jensen
polynomial:

```text
P_{d,n,lambda}(x)
  = sum_{j=0}^d binom(d,j) A_{n+j}(lambda) x^j.
```

This note isolates the exact finite-window object that connects the Jensen
criterion to total-positivity language. For every degree and shift, define the
binomially weighted Jensen window:

```text
B^{d,n,lambda}_j = binom(d,j) A_{n+j}(lambda), 0 <= j <= d.
```

The all-order target is not merely a signed-Hankel pattern for the unweighted
`A_k(lambda)`. It is:

```text
for every d >= 1 and n >= 0,
the finite sequence B^{d,n,0}_0,...,B^{d,n,0}_d is PF-infinity.
```

For a finite positive sequence, this is the finite Polya-frequency form of
the real-negative-root condition for its generating polynomial. Thus this
target is equivalent to all-degree/all-shift Jensen hyperbolicity, but it
phrases the missing bridge in Toeplitz total-positivity language.

## Exact Relation To Jensen

The infinite banded Toeplitz matrix generated by the finite window is:

```text
T_{d,n}(r,c) = B^{d,n,0}_{c-r}
```

with entries outside `0..d` set to zero. The target asks for every finite minor
of this banded Toeplitz matrix to be nonnegative, for every `d` and `n`.

Equivalently:

```text
sum_{j=0}^d B^{d,n,0}_j x^j
```

has only real nonpositive zeros for every `d,n`.

This is exactly the Jensen-window version of the all-order problem. It does
not prove the target; it removes ambiguity about what the signed-Hankel route
has to deliver.

## Contact With Signed Hankel

For `d = 2`:

```text
B_0 = A_n
B_1 = 2 A_{n+1}
B_2 = A_{n+2}
```

The discriminant condition is:

```text
B_1^2 - 4 B_0 B_2 >= 0
```

or:

```text
A_{n+1}^2 - A_n A_{n+2} >= 0.
```

That is exactly the `m = 1` signed-Hankel condition:

```text
-det [[A_n, A_{n+1}], [A_{n+1}, A_{n+2}]] >= 0.
```

So the signed-Hankel route has exact contact with Jensen at degree 2.

## Required Bridge Theorem

The theorem needed by this route is now:

```text
all-order shifted reshaped-Hankel sign-consistency for A_k(0)
+ Xi/Phi-specific analytic hypotheses
=> every binomially weighted Jensen window B^{d,n,0} is PF-infinity
```

or an equally strong direct theorem:

```text
for every d,n, the infinite banded Toeplitz matrix generated by
B^{d,n,0}_j = binom(d,j) A_{n+j}(0)
is totally nonnegative.
```

Either version would imply all Jensen polynomials are hyperbolic, and then the
remaining task would be the standard limiting passage to the required
Laguerre-Polya/Newman conclusion without assuming RH.

The current finite Toeplitz/PF certificate ledger checks the ordinary Taylor

```text

```

That is a separate coefficient-PF route. It should not be confused with the
binomially weighted Jensen windows:

```text
B^{d,n,lambda}_j = binom(d,j) A_{n+j}(lambda).
```
