```text
c_k(lambda) = A_k(lambda) / k!
F_lambda(s) = sum_{k >= 0} c_k(lambda) s^k
```

## Classical Theorem Anchors

The relevant external target is the one-sided Pólya-frequency sequence theorem of Aissen-Schoenberg-Whitney and Edrei, not a new local-zero-repulsion principle:

```text
Aissen, Schoenberg, Whitney, "On the generating functions of totally positive sequences I"
https://doi.org/10.1007/BF02786970

Edrei, "On the generating functions of totally positive sequences II"
https://doi.org/10.1007/BF02786971
```

Modern expository cross-check:

```text
Apoorva Khare, "Polya frequency sequences: analysis meets algebra"
https://www.imsc.res.in/~amri/algcomb/201112_khare.pdf
```

Thus the proof obligation should be written as:

```text
1. Prove the one-sided coefficient sequence c_k(0) = A_k(0)/k! is PF-infinity.
2. Apply the ASW/Edrei generating-function classification.
3. Verify that the resulting Laguerre-Polya conclusion is exactly the RH-normalized zeta/Jensen object, without assuming real zeros at lambda = 0.
```

## The Main Distinction

There are two different total-positivity objects:

```text
1. coefficient PF-infinity:
   Toeplitz minors of the sequence c_k = A_k/k!

2. kernel PF-infinity:
   translation minors of the de Bruijn-Newman heat kernel itself
```

These must not be merged.

Schoenberg's zeta condition is also a third object:

```text
inverse transform of 1 / Xi or 1 / Xi_1 is a Polya frequency function
```

That is an inverse-function condition, not a direct positivity theorem for the heat kernel moments.

## Source Anchors

### ASW / Edrei

ASW and Edrei characterize one-sided totally positive sequences by their generating functions. Groechenig summarizes the consequence we need: an entire function belongs to the restricted Laguerre-Polya class iff its Taylor coefficient sequence is a Polya frequency sequence.

Use this only in the following direction:

```text
proved PF-infinity of c_k
  => restricted Laguerre-Polya membership
```

Do not use it as:

```text
many finite minors checked
  => PF-infinity
```

unless a separate finite-to-infinite reduction theorem has been proved for this exact sequence.

### Schoenberg

Schoenberg's theorem relates Polya frequency functions to reciprocal Laplace transforms of Laguerre-Polya functions. This is extremely relevant, but it concerns:

```text
1 / Psi
```

not the original direct heat kernel moment sequence by itself.

### Direct Kernel PF Barrier

A 2026 arXiv preprint, "On the Polya Frequency Order of the de Bruijn-Newman Kernel", claims a certified PF5 failure for the direct de Bruijn-Newman kernel via a Toeplitz minor. Treat this as a serious warning sign for any proof path that tries to prove the heat kernel itself is PF-infinity.

This does not kill the `c_k = A_k/k!` coefficient PF route, because the objects are different:

```text
kernel translation minors     != coefficient Toeplitz minors of c_k
```

But it does add a hard gate:

```text
no manuscript claim may say the de Bruijn-Newman kernel is PF-infinity
without first resolving the claimed PF5 obstruction.
```

## What A Real Bridge Would Look Like

Any acceptable proof route needs one of the following.

### Bridge 1: Determinantal Integral Formula

Find an exact formula:

```text
det(c_{j_b - i_a})_{a,b=1}^m
  = integral of a manifestly nonnegative determinant/product
```

for every Toeplitz minor. This would be the cleanest route, because it converts all-order PF-infinity into a Cauchy-Binet / Andreief-type positivity statement.

Current blocker:

```text
no such formula is known for c_k = mu_{2k} k! / ((2k)! k!)
```

### Bridge 2: Stable Polynomial Approximation

Construct polynomial approximants:

```text
F_{lambda,n}(s) -> F_lambda(s)
```

locally uniformly, where every `F_{lambda,n}` has only real nonpositive zeros and coefficient convergence is compatible with the exact `c_k(lambda)`.

Current blocker:

```text
existing finite Jensen/root checks do not construct such approximants.
```

### Bridge 3: Continued Fraction Or Recurrence

Derive a continued-fraction, Stieltjes, Jacobi, or production-matrix representation whose coefficients are manifestly nonnegative and whose coefficient sequence is known to be PF-infinity.

Current blocker:

```text
the signed Hankel pattern for A_k is not the ordinary positive Hankel moment condition.
```

### Bridge 4: Heat-Flow Preservation

Prove that the heat-flow deformation preserves coefficient PF-infinity in the relevant lambda direction:

```text
c_k(lambda_1) PF-infinity
  => c_k(lambda_0) PF-infinity
```

for `lambda_0 <= lambda_1`, with a base point where PF-infinity is accessible.

Current blocker:

```text
the de Bruijn-Newman problem is exactly about avoiding a circular base point.
```
